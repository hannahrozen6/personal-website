# personal-website
This repository contains the source code and Markdown files for my personal website, which features my portfolio, resume, and contact information.

**Live Link:** [https://hannahrozenbaum.netlify.app/]

## Features
*   **Homepage:** A brief introduction about my background and technical interests.
*   **Portfolio:** A showcase of my data science/development projects, complete with code walkthroughs.
*   **Resume:** An updated, web-accessible version of my professional experience.
*   **Contact:** Direct links to reach out to me via email or professional socials.

## Built With
*   **Framework:** [Quarto](https://quarto.org/) (Markdown-based publishing system)
*   **Deployment & Hosting:** [Netlify](https://www.netlify.com/) (Connected via continuous deployment)

## 📁 Repository Structure
*   `_quarto.yml`: The configuration file controlling the website navigation, theme, and structure.
*   `index.qmd`: The homepage of the website.
*   `portfolio.qmd`: The project showcase page.
*   `resume.qmd`: My professional resume page.
